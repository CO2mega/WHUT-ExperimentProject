### 第五章

重构为使用 MySQL 数据库储存数据。

**注意：**为了防止密码泄露到 git 中，我将 SQL 配置文件提取到单独的 properties 文件内了。需要在项目 /src 目录下创建文件 `database.properties`，内容格式如下：

```properties
jdbc.driver=com.mysql.cj.jdbc.Driver
jdbc.url=jdbc:mysql://localhost:3306/<填写数据库名>
jdbc.username=<填写账号>
jdbc.password=<填写密码>
```

将所有功能补全。

**注意：**当时为了防止重构浪费时间，我把很多功能拖到现在才写。此时才将老师要求的所有功能逻辑完成。
